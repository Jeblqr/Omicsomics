# Omicsomics - 已实现功能清单

## ✅ 用户认证与授权

### 前端功能
- [x] **登录页面** - 完整的登录表单（邮箱 + 密码）
- [x] **注册页面** - 用户注册表单（姓名 + 邮箱 + 密码）
- [x] **登录/注册切换** - 在同一页面切换
- [x] **表单验证** - 客户端验证（必填项、邮箱格式、密码长度）
- [x] **错误处理** - 显示后端返回的错误信息
- [x] **自动登录** - 注册后自动登录
- [x] **登录状态保持** - 使用 localStorage 保存 token
- [x] **自动认证恢复** - 刷新页面后自动恢复登录状态
- [x] **路由保护** - 未登录自动跳转到登录页
- [x] **登出功能** - 清除登录状态并跳转
- [x] **用户信息展示** - 顶部栏显示用户名
- [x] **美化的 UI** - 现代化设计、渐变色、卡片样式

### 后端 API
- [x] POST `/api/v1/register` - 用户注册
- [x] POST `/api/v1/login/access-token` - 用户登录
- [x] JWT Token 认证
- [x] 密码加密存储（pbkdf2_sha256）

## ✅ 技术架构

### 前端技术栈
- [x] **React 18** - UI 框架

- [x] **Vite** - 构建工具
- [x] **React Router** - 路由管理
- [x] **React Query** - 数据获取和缓存

- [x] **JWT** - Token 认证
